# Muhammad Fawad — DMCA Takedown Portfolio Site

A static, dependency-free portfolio site (HTML/CSS/JS only) ready for GitHub Pages.

## Files
- `index.html` — all page content and sections
- `css/style.css` — design system, layout, responsiveness, animations
- `js/script.js` — mobile nav, FAQ accordion, scroll reveal

## Before you launch — replace these placeholders

1. **Contact form** (`index.html`, `<form ... action="https://formsubmit.co/your-email@example.com">`)
   Replace `your-email@example.com` with your real email. FormSubmit.co is free and needs
   no backend — the first submission sends a confirmation email you must click to activate it.

2. **Contact info block** — update the Email / Fiverr / Upwork lines in the Contact section.

3. **Social links** — the four `href="#"` links in the footer (Fiverr, Upwork, LinkedIn, X) — add your real profile URLs.

4. **Testimonials** — the four sample quotes are placeholders. Swap in real client feedback before launch (a note is included on the page reminding you, remove it once replaced).

5. **Case file number / stats** in the hero (`#MF-2026-0417`, `500+`, etc.) — adjust to your real numbers if you'd like exact figures.

## Deploying to GitHub Pages

1. Create a new GitHub repository (e.g. `dmca-portfolio`).
2. Push these files to the repo root (keep the `css/` and `js/` folder structure).
3. In the repo: **Settings → Pages → Source → Deploy from branch → `main` / `root`**.
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

No build step, no dependencies — it's plain static files, so this is the entire process.

## Notes
- Fonts (Space Grotesk, IBM Plex Sans, IBM Plex Mono) load free from Google Fonts via CDN.
- All icons are inline SVG — no icon library dependency, keeps the page fast.
- Reduced-motion is respected for users with that OS preference.
